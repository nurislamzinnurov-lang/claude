# Калькулятор

Простой веб-калькулятор на HTML/CSS/JavaScript без зависимостей.

## Запуск

Откройте `index.html` в браузере — никакой сборки или сервера не требуется.

## Возможности

- Базовые операции: `+`, `−`, `×`, `÷`
- Смена знака `±`, процент `%`, сброс `C`
- Поддержка клавиатуры: цифры, `.` / `,`, `+ - * /`, `Enter` / `=`, `Backspace`, `Esc`, `%`
- Защита от деления на ноль (показывает `Error`)
